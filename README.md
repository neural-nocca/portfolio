# Мої проєкти — портфоліо з GitHub

Статична сторінка, яка **автоматично** підтягує твої репозиторії з GitHub
і показує їх списком. Нічого вписувати вручну не треба: створила репозиторій —
він сам з'явився у списку.

## Як подивитися

Просто відкрий `index.html` подвійним кліком у браузері.

> Якщо смужки мов не з'явилися — це GitHub обмежує **60 запитів/год** для
> неавторизованих. Зачекай годину або опублікуй сайт (нижче).

## Файли

- `index.html` — структура сторінки
- `styles.css` — оформлення (палітра вгорі, у блоці `:root`)
- `app.js` — логіка: тягне дані з GitHub
- `particles.js` — частинки на фоні (налаштування у `CONFIG` вгорі)

## Як опублікувати безкоштовно (GitHub Pages)

1. Створи на GitHub новий репозиторій, напр. `portfolio`.
2. Залий туди всі файли цієї папки.
3. У репозиторії: **Settings → Pages → Branch: main → Save**.
4. За хвилину сайт буде за адресою `https://neural-nocca.github.io/portfolio/`.

## Як змінити

- **Нік** — атрибут `data-username` у верхньому теґу в `index.html`.
- **Кольори** — блок `:root { ... }` на початку `styles.css` (усі в одному місці).
- **Частинки на фоні** — блок `CONFIG` на початку `particles.js`
  (колір, кількість, розмір, швидкість, прозорість).
- **Текст «про себе»** — змінна `ABOUT` на початку `app.js`. Якщо порожня —
  візьметься поле **Bio** з GitHub-профілю.
- **Куди веде стрілка › на телефоні** — змінна `HEADER_LINK` в `app.js`
  (порожньо = профіль GitHub; можна вписати `mailto:пошта` тощо).
- **Опис проєкту на картці** береться з поля **About → Description** самого
  репозиторію на GitHub. Порожнє → на картці «Без опису».

## Що показується

- **Десктоп, ліворуч (нерухомо):** аватар, нік, ім'я, «про себе»,
  лічильник проєктів, посилання на профіль.
- **Телефон:** угорі нерухома шапка (аватар · нік+ім'я · стрілка ›),
  під нею картки в одну колонку.
- **Картки:** назва, опис, ⭐ зірки, дата оновлення і смужка
  співвідношення мов у проєкті.
- Згори: пошук і сортування (нові / за зірками / за назвою).
- Форки й архівовані репозиторії приховані.
